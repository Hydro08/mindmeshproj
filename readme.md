===> FLASHCARD WEB APP <===

A simple web app for creating and reviewing flashcards. All data is stored locally in your browser using localStorage, so no backend is required.

===> FEATURES <===

• Create, edit, and delete decks
• Add, edit, and delete flashcards
• Review flashcards in quiz mode
• All data is saved automatically in your browser localstorage

===> HOW TO RUN <===

• Open the folder in vs code
• Install the live server extension (if you don’t have it)
• Right-click login.html and select "open with live server"
• The website will open in your browser, ready to use

# Optional: To reset all data, open the browser console (F12 or Ctrl+Shift+I) and type: localStorage.clear()

===> USAGE <===

• Click "create deck" and input name and category
• Click the created deck in the dashboard
• Click the "create flashcard" and input question and answer
• Click "create flashcard" (button) when done

===> NOTES <===

• Data only exists on the current browser and device
• Switching browsers or devices will not carry over your decks and flashcard
• No server or backend setup is required
